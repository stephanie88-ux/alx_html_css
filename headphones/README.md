# Headphones Landing Page

A modern, responsive landing page for premium headphones based on the Figma design.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet and mobile devices
- **Modern UI**: Dark theme with coral accent colors
- **Smooth Animations**: Floating hero image, hover effects, and transitions
- **Clean Code**: Semantic HTML5 and organized CSS

## Project Structure

```
headphones-landing/
├── index.html          # Main HTML file
├── style.css           # Stylesheet with all styling
├── README.md           # Project documentation
└── assets/
    └── images/         # Folder for images (add your headphones image here)
```

## Getting Started

1. Open `index.html` in your web browser
2. Add a headphones image to `assets/images/headphones.png` for the hero section

## Design Elements

- **Color Scheme**:
  - Background: Dark navy blue (#1a2332, #2a3d54)
  - Accent: Coral red (#EF4444)
  - Text: White and light gray

- **Sections**:
  - Header with navigation
  - Hero section with CTA button
  - Features section with 4 icon cards
  - Statistics section with pentagon shapes
  - Contact section
  - Footer

## Responsive Breakpoints

- Desktop: 1024px and above
- Tablet: 768px - 1024px  
- Mobile: Below 768px
